**Tropical Cyclone Risk Analysis TCRA**


Overview
--------
**TCRA** is a Python package that allows performing scenario-based tropical cyclone damage to buildings and electrical poles.


Dependency
----------
* numpy
* pandas
* folium
* matplotlib
* scipy
* fiona
* collections
* concurrent.futures
* past.builtins
* typing
